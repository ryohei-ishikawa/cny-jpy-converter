# 元換算 — CNY / USD → JPY 即換算（Chrome 拡張）

中国サイト（1688・淘宝など）で価格などの数字をドラッグ選択すると、その場で
**元→円・USD→円をまとめて日本円に換算**してふきだし表示する Chrome 拡張機能（Manifest V3）。

## 特長
- 数字を選択 → 🇨🇳 元→🇯🇵 円 と 🇺🇸 USD→🇯🇵 円 を同時にふきだし表示
- 各行クリックで円の値をコピー（コピーは常に正確な整数値）
- **桁が大きいときは `1.23億円` `1234万円` のように万/億/兆でコンパクト表示**（ホバーで正確な値）
- **「1万8538」「二億三千四百五十六万」「123,456」「１２３４」など、漢数字・全角・漢字単位混じりの様々な表記**にも対応
- 為替レートは無料API（open.er-api.com／フォールバックに frankfurter）から自動取得・6時間ごと更新
- 仕入れ用の **手数料・利益の上乗せ%**、選択時の自動表示ON/OFF、中国系サイト限定動作などを設定可能

## インストール（開発者モード）
1. `chrome://extensions` を開く
2. 「デベロッパーモード」をON
3. 「パッケージ化されていない拡張機能を読み込む」→ このフォルダを選択

## 構成
- `manifest.json` … MV3 設定
- `background.js` … 為替レート取得・キャッシュ（service worker）
- `content.js` / `content.css` … 選択検知・ふきだし表示
- `popup.html` / `popup.js` / `popup.css` … レート表示・電卓・設定
